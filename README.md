## Technologies Used

- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JSON Web Tokens (JWT)

## Getting Started

### Prerequisites

- Node.js 
- npm or yarn
- MongoDB

### Clone the Repository

```bash
git clone https://github.com/your-username/task-management-app.git
cd task-management-app

npm install

and then run
